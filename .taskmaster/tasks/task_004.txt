# Task ID: 4
# Title: 분석 중 로딩 화면 UI 개발
# Status: pending
# Dependencies: 3
# Priority: medium
# Description: PDF 업로드 후 분석이 진행되는 동안 표시되는 로딩 화면 UI를 구현합니다.
# Details:
1. 로딩 스피너 또는 프로그레스 바 컴포넌트 생성: Shadcn UI의 Progress 컴포넌트를 활용하여 원형 스피너와 선형 프로그레스 바 구현
2. 분석 진행 상태를 나타내는 단계별 표시 UI: "PDF 업로드 중" → "텍스트 추출 중" → "AI 분석 중" → "결과 생성 중" 단계별 상태 표시
3. 예상 소요 시간 표시 기능: 파일 크기 기반 예상 시간 계산 로직 구현 및 실시간 카운트다운 표시
4. 분석 취소 버튼 UI (기능은 나중에 구현): 취소 버튼 컴포넌트 생성 및 비활성화 상태로 배치
5. 로딩 중 애니메이션 효과 추가: CSS 애니메이션과 Framer Motion을 활용한 부드러운 로딩 애니메이션 구현
6. 사용자에게 분석 과정을 설명하는 텍스트 표시: 각 단계별 설명 텍스트와 도움말 메시지 표시
7. 반응형 디자인 적용: 모바일, 태블릿, 데스크톱 화면에 최적화된 레이아웃 구현
8. 로딩 완료 시 분석 결과 화면으로 전환하는 UI 연결: React Router를 사용한 자동 페이지 전환 로직 구현
9. 로딩 상태 관리를 위한 Context API 또는 상태 관리 라이브러리 활용
10. 에러 발생 시 에러 메시지 표시 및 재시도 옵션 제공
